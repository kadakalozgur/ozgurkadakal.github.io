VaultNet - Güvenli Kasa API

Bu proje C# dilinde .Net Core ile kodlanmış bir Vault API projesidir.

  • 🔐 Kimlik Doğrulama ve Güvenlik
    • 🛡️ JWT Authentication: Token tabanlı güvenli giriş sistemi.
    • 🔑 Şifre Hashleme: Şifreler BCrypt ile güvenli şekilde saklanır.
    • 🧾 Claim Tabanlı Kimlik: Kullanıcı bilgileri JWT üzerinden yönetilir.
    • 🔒 Veri Şifreleme: Hassas veriler AES-256 (Data Protection API) ile şifrelenir.
  • 📦 Kasa Yönetimi
    • ➕ Kayıt Ekleme: Şifre, not vb. hassas verileri saklama.
    • ✏️ Güncelleme: Mevcut kayıtları düzenleme.
    • ❌ Silme: Yetkili kullanıcı tarafından güvenli silme.
    • 📂 Verilerim: Sadece kullanıcıya ait verileri listeleme.
  • 👤 Kullanıcı Sistemi
    • 📝 Kayıt Ol: Yeni kullanıcı oluşturma.
    • 🔐 Giriş Yap: JWT token alma.
    • 📧 Benzersiz E-posta Kontrolü
    • 👁️ Profil Bilgisi: Claim üzerinden kullanıcı bilgisi alma.
  • ⚙️ Backend Mimarisi
    • 🧩 Katmanlı Mimari: Controller → Service → Data katmanı ayrımı.
    • 🔄 Dependency Injection
    • 🗄️ Entity Framework Core
    • 🌐 Global Exception Handling
    • ⚡ Async Programlama
